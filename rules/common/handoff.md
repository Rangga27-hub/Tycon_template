# Common: Handoff

This project is built by multiple AIs in turns. Handoff is manual via `.md` files.

- Read `docs/meta/CONTINUE.md` at the start of every session.
- Update `CONTINUE.md` + `docs/engineering/PROGRESS.md` before stopping.
- Write CONTINUE.md for an AI that knows nothing — name files, functions, lines.
- Chat history does NOT transfer. Only files do.
- Mark unfinished work with `// TODO(handoff):` and note it in CONTINUE.md.
- Run `docs/checklists/handoff.md` before ending.
