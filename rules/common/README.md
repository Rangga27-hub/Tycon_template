# Rules

Always-follow engineering rules. Structure:

- `common/` — universal, every project type (always active).
- Type-specific rules are created at INIT when needed (e.g. `rules/ml/`, `rules/web/`)
  and listed in the active `stacks/<type>.md`.

These are guardrails. AGENTS.md Source-of-Truth Hierarchy outranks them only where noted.
